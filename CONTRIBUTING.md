# Contributing

TogeDog은 교육 프로젝트의 공개 포트폴리오 저장소입니다.

## 브랜치

- `main`: 검증된 공개본
- `docs/*`: 문서 개선
- `feat/*`: 기능 추가
- `fix/*`: 오류 수정

## 커밋 예시

- `feat: add walk danger websocket event`
- `fix: prevent duplicate TTS alerts`
- `docs: document Firebase repository schema`
- `security: remove embedded credential`

## Pull Request 체크

- 실제 비밀값과 개인 정보가 포함되지 않았는지 확인
- 생성 파일과 캐시가 제외되었는지 확인
- 구현되지 않은 기능을 완료 기능처럼 표현하지 않았는지 확인
- 실행 방법 또는 변경 문서를 함께 갱신
